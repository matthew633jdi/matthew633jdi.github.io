# 기능 설계

- 사용자 기능
  - 사용자 등록
  - 사용자 조회
- 문제 기능
  - 문제 등록
  - 문제 수정
  - 문제 조회
- 복습 기능
  - 복습 등록
  - 복습 수정
  - 복습 조회
- 기타 요구사항
  - 복습할 날짜가 현재 날짜보다 이전일 수 없다.
  - 기존에 등록한 문제 중복 불가
  - 로그인 시 당일 날짜의 복습 문제를 리스트업
  - 각 문제를 클릭하면 문제 상세 정보와 복습 정보를 확인
  - 복습을 하지 않으면 계속 출력됨
  - 복습 날짜가 오래된 문제부터 리스트업(복습하지 않은 날짜 순)
  - 유형별 조회
  - 레벨별 조회
  - 모든 문제 리스트업
